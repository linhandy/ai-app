② 部署方案（极低成本）

🚀
Vercel 托管（免费）
你已有其他项目在上面。Next.js 项目 git push 即自动部署，支持自定义域名，免运维。
🌐
域名（约55元/年）
建议用 .cn 域名，更适合国内用户信任感。如 ai-ticai.cn / zhuti-ai.cn 等。去阿里云/腾讯云注册。
🗄️
Supabase 数据库（免费额度够用）
你已在 hourse-designer 中用过。存用户账号、使用次数、订单信息，免费额度对早期产品完全够用。
⚠️
备案问题（重要）
国内域名+国内用户访问，理论上需要ICP备案。Vercel 部署在境外服务器，短期内可先跑流量，验证付费意愿后再决定是否迁移至国内服务器做备案。

③ 运营策略（无需主动推广）

主动传播（一次性，极低成本）

  • 往 2-3个 教师微信群/QQ群发一份示例试卷截图
  • 在知乎回答「如何快速出试卷」类问题，自然带链接
  • 在闲鱼挂一个「帮出试卷」服务引流（你已有经验）

持续自然增长（产品内置）

  • 免费PDF水印 → 同事看到 → 自然裂变
  • SEO：百度收录 「AI出题」「在线组卷」关键词
  • 生成后引导「分享到微信群」

💡 关键认知：教师群体极度依赖同事推荐，一个群里一个人用了觉得好，整个群都会试用。你只需触达第一批种子用户，后续传播是自动的。

④ 收费方案（个人开发者最优解）

收款工具推荐：虎皮椒支付

xunhupay.com — 个人开发者无需营业执照，接入微信支付+支付宝，提供API，接入成本极低（1-2天开发）。

适合自助付费，用户自己买，不需要你手动发货。

备选：爱发电订阅

afdian.net — 零开发成本，创建订阅套餐，用户直接在爱发电付款，你在后台发放兑换码。适合早期快速验证付费意愿。

缺点：需要手动发放，不够自动化。
