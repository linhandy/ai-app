1A · 图片存储迁移

目标：Cloudflare R2

  • 上传图片 → R2，返回公开 URL

1B · 免费体验层

  • Redis 或 SQLite 记录 IP 使用次数

  • 用完3次 → 显示充值弹窗

💡 7天重置防过于宽松，又不至于永久封锁
