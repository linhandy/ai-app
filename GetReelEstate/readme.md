# 项目名称：AI 房产短视频生成器 (AI Real Estate Video Generator)

## 1. 项目背景与对标
我正在开发一个面向海外房产经纪人 (Realtors) 的 Micro-SaaS 工具。
- **对标产品**：https://www.listing2video.com/
- **核心功能**：用户只需输入房源图片和一段房源描述文本，系统就能自动生成一个 30 秒的竖屏 (9:16) 房产营销短视频。
- **视频要求**：必须包含 AI 配音（激情的口播）、图片缓慢缩放的运镜效果 (Ken Burns effect)，以及屏幕中央醒目的动态词级字幕 (Hormozi 风格)。

## 2. 核心工作流 (The Pipeline)
整个自动化流水线包含以下步骤：
1. **输入**：接收 5-7 张高清房源图片和一段描述（例如：价格、几室几厅、卖点）。
2. **文案生成 (LLM)**：调用大模型，将房源描述改写成 60-80 个英文单词、适合短视频口播的营销文案。
3. **语音合成 (TTS)**：将生成的文案转成极具表现力的 `.mp3` 语音文件。
4. **字幕时间戳提取 (ASR)**：通过音频识别获取 Word-level（词级别）的时间戳。
5. **视频渲染**：将图片（加转场动效）、配音和动态字幕合并压制成最终的 `.mp4` 视频文件。

## 3. 技术栈
- **前端**：Next.js (App Router), Tailwind CSS, React。
- **后端**：Next.js API Routes (如果渲染太重，可单独开 Node.js 脚本/服务)。
- **AI 接口配置 (非常重要)**：所有 AI 能力（LLM / TTS / 语音转文本）全部调用我个人的网关：
  - Base URL: `https://zenmux.ai/api/v1`
  - 使用 OpenAI 兼容的 SDK 进行调用，只需替换 Base URL 和 API Key。
- **视频渲染引擎**：请使用 `FFmpeg` (通过 Node.js 的 `fluent-ffmpeg`) 或者 `Remotion` (基于 React 的视频渲染)。请你作为专家，为我选择一个最适合当前 MVP 的方案并给出理由。
