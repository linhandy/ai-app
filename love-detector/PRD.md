## 一、产品概述

### 1.1 产品定位

"恋爱测谎仪"是一款面向年轻情侣、单身人群的**社交趣味工具型 App**。用户可以通过语音、文字、微信记录等多种方式，对恋人或自己进行"可信度检测"，获得 AI 分析报告，制造话题、增进亲密感，同时满足用户对伴侣行为的好奇心。

### 1.2 设计关键词

> 好玩 · 刺激 · 高颜值 · 彩虹感 · 游戏化 · 社交传播

### 1.3 目标用户

| 用户类型 | 场景 |
|---------|------|
| 情侣用户 | 怀疑 TA 在撒谎，想当面测一测 |
| 单身用户 | 和暧昧对象玩，通过测试破冰 |
| 闺蜜/基友 | 互测好友，看谁说谎最多 |
| 自测用户 | 想知道自己有多真诚/多能骗人 |

## 二、核心功能模块

### 2.1 测谎模式（★ 核心）

#### 模式一：测 TA（给他人测）
- 当面出题，对方大声回答
- 实时录音分析声纹特征
- 给出逐题可信度 + 最终综合报告

#### 模式二：测自己
- 自我审问模式（对着手机自言自语）
- 更轻松的问题库（自省向）
- 结果定性为"自我诚实度"评分

#### 模式三：微信记录测谎（新增 ✨）
- 用户粘贴微信对话文字
- 上传微信语音（.amr / .mp3）
- AI 分析文字逻辑一致性 + 语义情感偏差
- 给出"这段聊天有多少水分"报告

---

### 2.2 输入方式

| 输入方式 | 描述 | 适用模式 |
|---------|------|---------|
| 🎙️ 实时录音 | 对方当面作答，App 录音分析 | 测 TA / 测自己 |
| ⌨️ 文字回答 | 用户手动输入对方的回答 | 测 TA / 微信记录 |
| 💬 微信文字粘贴 | 粘贴微信聊天记录文字 | 微信记录模式 |
| 🔊 微信语音上传 | 转发/上传微信语音文件 | 微信记录模式 |
| 📋 截图上传（可选）| 上传聊天截图，OCR 识别文字 | 微信记录模式 |

---

### 2.3 AI 分析维度

#### 实时录音分析
- **声纹特征**：音调波动、停顿频率、语速变化
- **情绪指数**：紧张 / 平静 / 兴奋
- **关键词检测**：模糊词、回避词、过度解释

#### 文字分析
- **逻辑一致性**：前后矛盾检测
- **情感倾向**：正面 / 中性 / 回避
- **细节丰富度**：回答越细节=越可信 or 越编

#### 综合可信度
- 0-100 分，带颜色区间：
  - 85-100：💚 非常可信（放心了）
  - 60-84：💛 基本可信（留个心眼）
  - 35-59：🧡 存疑（有点水分）
  - 0-34：❤️ 高度可疑（可能在骗你）

---

### 2.4 问题库系统

#### 题库分类
| 分类 | 示例问题 |
|------|---------|
| 💑 忠诚度 | "昨天晚上你真的在加班吗？" |
| 📱 手机隐私 | "你手机里有什么不能给我看的吗？" |
| 💰 财务诚实 | "你这个月花了多少钱在游戏上？" |
| 😍 感情深度 | "你有没有喜欢过别人但没告诉我？" |
| 🎭 小谎言 | "你觉得我今天好看吗？" |
| 🔍 自我认知 | "你真的爱我多少分？" |

#### 题库规则
- 每次测谎默认 5 题，可选 3 / 5 / 10 题
- 支持用户自定义问题（最多 3 题/次）
- 问题有难度分级：🌶 温和 / 🌶🌶 犀利 / 🌶🌶🌶 灵魂拷问

## 四、视觉设计规范

### 4.1 主色调

| 角色 | 颜色 | 用途 |
|------|------|------|
| 主渐变 | `#FF6B9D → #C44DFF → #7B4DFF` | 按钮、进度条、主视觉 |
| 真话色 | `#00D2A0` 青绿色 | 可信状态 |
| 谎言色 | `#FF4E6A` 玫红色 | 可疑状态 |
| 中性色 | `#FFB800` 琥珀色 | 存疑状态 |
| 背景 | `#FAFAFA`（亮色） / `#0D0B1F`（暗色） | 整体背景 |

### 4.2 视觉风格

- **风格**：糖果色系 + 毛玻璃卡片 + 霓虹发光效果
- **字体**：标题用圆润加粗字体，内容用常规字体
- **动画**：心跳动画、声波动画、结果揭晓动画（类刮刮乐）
- **图标**：彩色渐变图标，表情包风格插图

### 4.3 关键动效

| 场景 | 动效 |
|------|------|
| 首页等待 | 心率/脑波跳动动画 |
| 录音中 | 彩色声波可视化 |
| 分析中 | 旋转雷达扫描动画 |
| 结果揭晓 | 刮刮卡/爆炸式展开 |
| 判定真话 | 绿色光晕扩散 |
| 判定谎言 | 红色震动效果 |

## 五、游戏化机制

### 5.1 成就系统
- 🔍 "福尔摩斯"：测出 10 次谎言
- 💚 "信任大师"：连续 5 次全真话
- 👁️ "第六感"：人工判断与 AI 一致率 90%+
- 🎯 "灵魂拷问"：完成 3 次🌶🌶🌶难度测谎

### 5.2 分享机制
- 结果自动生成**精美分享卡**（仿测谎仪报告格式）
- 微博/朋友圈一键分享（引流裂变）
- 支持生成"挑战 TA 来测"的邀请海报

### 5.3 每日彩蛋
- 每天一道"今日灵魂之问"（限时题目）
- 完成获得积分，积分可解锁稀有题库

## 六、技术方案（前端视角）

### 6.1 技术栈建议

| 层 | 技术 |
|----|------|
| 框架 | React Native（跨平台）或 微信小程序 |
| 状态管理 | Zustand / Redux |
| 动画 | Lottie（复杂动效）+ CSS Animation |
| 录音 | Web Audio API / RN 原生录音 |
| AI 分析 | 调用后端 API（Claude / 语音识别 API）|
| 分享卡生成 | Canvas / html2canvas |

### 6.2 关键前端能力

```
1. 录音模块
   - 启动/停止录音
   - 实时音量波形可视化
   - 录音文件上传

2. 文字输入模块
   - 富文本粘贴（微信截图 OCR）
   - 长文本分析

3. 分析动画
   - 进度轮询 + 加载动效
   - 结果分段渲染

4. 分享卡生成
   - Canvas 绘制报告卡
   - 保存到相册
```
